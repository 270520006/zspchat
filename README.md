## 框架/技术选用

​	传统的聊天工具都需要一个服务器，然后再建立客户端，对服务器端进行发送消息，然后让服务器端分发消息到指定的用户上。下面我们说一下选择使用的框架/技术

* netty：使用netty的socket通信比起没有选择器的低效BIO和代码复杂、bug多的NIO来说，非常的惬意。netty是基于reactor模式下开发的一个NIO框架，整体更为规范。并且由于netty在不断的升级和迭代，目前API使用也特别舒服。
